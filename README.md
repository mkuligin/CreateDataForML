# Генератор тестовых данных для машинного обучения

## Описание программы

Этот скрипт на Python предназначен для генерации синтетических тестовых данных, имитирующих сетевой трафик. Данные могут использоваться для обучения и тестирования моделей машинного обучения, предназначенных для обнаружения атак в сети (например, классификация трафика как "normal" или "attack").

Скрипт анализирует предоставленный пример данных и генерирует новые строки, соблюдая следующие правила:
*   **Диапазоны значений:** Все числовые значения генерируются строго в пределах минимальных и максимальных значений, присутствующих в исходном примере для каждого типа метки (`normal` или `attack`).
*   **Структура:** Выходной файл сохраняет структуру и названия колонок исходного примера.
*   **Баланс классов:** Данные генерируются с учетом распределения классов, присутствующего в примере (примерно 60% нормального трафика и 40% атак).

## Назначение

*   Создание больших наборов данных для обучения моделей ML без использования реальных чувствительных данных.
*   Тестирование конвейеров обработки данных (data pipelines).
*   Демонстрация работы алгоритмов классификации.

## Структура данных

Генерируемый файл `network_traffic.csv` содержит следующие колонки:

| Колонка | Описание | Тип данных |
| :--- | :--- | :--- |
| `duration` | Длительность соединения (секунды) | int |
| `src_bytes` | Количество байт от источника | int |
| `dst_bytes` | Количество байт получателю | int |
| `wrong_fragment` | Количество фрагментов с ошибками | int |
| `urgent` | Количество срочных пакетов | int |
| `hot` | Количество горячих индикаторов | int |
| `num_failed_logins` | Количество неудачных попыток входа | int |
| `count` | Общее количество соединений | int |
| `label` | Метка класса: `normal` или `attack` | string |

## Требования

*   Python 3.x
*   Стандартные библиотеки: `csv`, `random` (дополнительная установка не требуется).

## Пример запуска

1.  Убедитесь, что файл скрипта `generate_data.py` находится в текущей директории.
2.  Запустите скрипт из командной строки:

```bash
python generate_data.py
```

3.  После выполнения в той же директории появится файл `network_traffic.csv` containing 1000 строк сгенерированных данных.

## Пример вывода (первые строки CSV)

```csv
duration,src_bytes,dst_bytes,wrong_fragment,urgent,hot,num_failed_logins,count,label
15,1400,2800,0,0,0,0,9,normal
42,7500,600,1,0,2,3,22,attack
...
```
